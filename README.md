**PROBLEM STATEMENT**
Students and beginners in electronics often choose LED resistors by guessing or copying values from other circuits. This can cause the LED to draw too much current or appear too dim. A simple software tool is needed to calculate the correct resistor value and provide basic safety feedback before the circuit is built.
**AIM OF THE PROJECT**
Students and beginners in electronics often choose LED resistors by guessing or copying values from other circuits. This can cause the LED to draw too much current or appear too dim. A simple software tool is needed to calculate the correct resistor value and provide basic safety feedback before the circuit is built.
**OBJECTIVES OF THE PROJECT**
•Accept supply voltage, LED forward voltage and desired LED current from the user.
• Validate all input values before performing calculations.
• Calculate the recommended current limiting resistor value using Ohm’s Law.
• Calculate the estimated resistor power rating required for the circuit.
• Classify the design as safe, invalid or requiring review.
• Display a clear LED circuit design report.
• Save the calculated report into a text file.
